## Run Locally

Clone the project

```bash
  git clone https://github.com/alirezanezami1/Portfolio.git
```

Go to the project directory

```bash
  cd portfolio
```

Install dependencies

```bash
  npm install
```

Start the development server on http://localhost:3000:

```bash
  npm run dev
```
